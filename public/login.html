How to Get Your Access Token

×

Ohio State Canvas

  1. Go to canvas.osu.edu and log in
  2. Click on your profile picture in the top right
  3. Select Settings from the dropdown menu
  4. Scroll down to Approved Integrations section
  5. Click New Access Token
  6. Enter a purpose like "Akorn"
  7. Set expiration to "Never" or choose a future date
  8. Click Generate Token
  9. Copy the token immediately (you won't see it again!)
  10. Paste it in the Access Token field above

Blackboard

  1. Log into your Blackboard account
  2. Go to Tools â†’ Personal Information
  3. Click Personal Settings
  4. Select Developer Settings
  5. Click Create Application
  6. Fill in the required information
  7. Copy the Application Key

D2L Brightspace

  1. Log into your D2L Brightspace account
  2. Go to Account Settings
  3. Click API Access
  4. Click Create New Token
  5. Enter a name for your token
  6. Set permissions to Read Only
  7. Click Generate Token
  8. Copy the token immediately

Note: Keep your access token secure and don't share it with anyone. You can revoke it anytime from your LMS settings.
